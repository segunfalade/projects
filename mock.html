The method

The First-Problem Map

We don't build a five-year plan in the first session. We find the single decision that, once made, makes a dozen others fall away — and we make that one. Then the next, in the order your own situation demands rather than the order a generic program assumes.

It works the same whether you're leaving a twenty-year career or finally starting the one you postponed — only the first problem changes.
